# CLAUDE.md — Strona Elizée (Wizażystka)

## Zasady krytyczne (zawsze aktywne)

- Język z użytkownikiem: **polski**
- Tech stack: **HTML5 + CSS3 + vanilla JS** — zero frameworków, zero zewnętrznych bibliotek
- Informacje prawdziwe o Elizie z `rules/content.md` są **nienaruszalne**
- Każda zmiana wizualna musi używać tokenów z `rules/design.md`
- Przed edycją **zawsze przeczytaj plik** narzędziem Read
- Edytuj przez `Edit` (nie przepisuj całych plików)
- Przed dodaniem `<img src="images/...">` — sprawdź prawdziwe nazwy plików w `images/` (mogą mieć inne rozszerzenia niż się wydaje)
- Można usuwać sekcje HTML gdy użytkownik o to prosi

## Indeks reguł

| Plik               | Zawartość                                        |
|--------------------|--------------------------------------------------|
| `rules/design.md`  | Tokeny kolorów, typografia, przyciski, breakpointy |
| `rules/content.md` | Bio Elizy, usługi, FAQ, ton copywritingu         |
| `rules/coding.md`  | BEM, struktura plików, konwencje HTML/CSS/JS     |
| `rules/workflow.md`| Kolejność sekcji, nowe sekcje, czego nie robić   |
